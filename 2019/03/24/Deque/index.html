[Leetcode 199] Using Double-ended Queue(Deque) to Solve Right-side View Problem of Binary Tree / 使用双向队列解决二叉树右视图问题

Posted on 2019-03-24 | In Leetcode

Given a binary tree, imagine yourself standing on the right side of it, return the values of the nodes you can see ordered from top to bottom.

Example:

Solution Code

Using a deque to traverse the tree.

Take test case [4, 3, 6, 1, null, 5, null, null, 2] as an example :

And the deque changes like that :

[4], prev = 4, cur = 4;

[3, 6], prev = 6, cur = 6;

[1, 5], prve = 5, cur = 5;

[2], prve = 2, cur = 2.

Then the result is [4, 6, 5, 2].

This solution using deque to deal every level of the tree, finding the last element of an level, which is the right-side node we are finding.

用双向队列遍历每一层，每一层的最右边元素就是要找的右视图结点。
